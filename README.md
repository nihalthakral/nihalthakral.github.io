## 📌 Yeh Repo Kya Hai?

Yeh **`nihalthakral.github.io`** ka source code hai — meri GitHub Pages personal site.

> *"Aapko convince karna nahi hai. Sirf ek mauka chahiye."*

Ek clean, honest page jo directly client se baat karti hai — bina bakwaas ke.
GitHub Pages ke through automatically deploy hoti hai jab bhi `main` branch pe push karo.
